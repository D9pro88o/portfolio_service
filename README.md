# Portfolio README

## Overview

This is a professional single-page portfolio built using **HTML, CSS, and JavaScript**. The design is clean, modern, and lightweight, offering smooth navigation and clear content structure.

The portfolio includes:

* A fixed navigation bar
* An introduction section
* A courses section
* Smooth scrolling interaction
* Responsive and minimal layout

---

## Features

### ✅ Clean & Modern UI

The page uses a minimal design approach with soft shadows, smooth transitions, and a balanced layout.

### ✅ Smooth Navigation

Navigation links allow the user to jump to sections using smooth scrolling powered by JavaScript.

### ✅ Fully Responsive

The layout is optimized for various screen sizes, making it suitable for desktop and mobile devices.

### ✅ Organized Sections

* **Introduction**: Brief overview of the developer.
* **Courses**: List of completed or learned courses.
